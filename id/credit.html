If we can write this code it is only by standing on the shoulders of giants

Panada dikembangkan dan belajar dari banyak aplikasi dan framework yang sudah sangat populer saat ini.

  • Yii

    Yii, sebuah framework yang kaya feature seperti MVC, DAO/ActiveRecord, I18N/L10N, caching, authentication dan role-based access control dan scaffolding. Beberapa konsep Active Record Yii digunakan di Active Record Panada.

  • CodeIgniter

    CodeIgniter Merupakan Framework PHP yang dikembangkan oleh Ellislab, Inc. Panada meminjam konsep CodeIgniter dalam hal kemudahan penggunaan dan manajemen pengelompokan file Model, View dan controller (MVC).

  • Kohana

    Kohana adalah Framework HMVC yang berbasiskan PHP 5. Kohana menginspirasi Panada dalam hal kemudahan me-load sumber daya secara cepat dan mudah.

  • WordPress

    WordPress merupakan Open Source CMS Blog yang paling populer saat ini.

  • ezSQL

    ezSQL adalah open source PHP database class yang dikembangkan oleh Justin Vincent.

  • PHPMailer

    PHPMailer adalah open source PHP class yang membantu para developer dalam pengiriman email.
